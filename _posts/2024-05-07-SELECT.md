select : 결과창에 보여줄 값을 지정

``` sql
select 컬럼명 from 테이블명 ;
```

```sql
select distinct 컬럼명  -- 중복되지 않은 속성값만 출력
	컬럼명 [as] 별칭 -- 별칭이 한글일 경우 "" 필요
from 테이블명 별칭 ;


/*
테이블의 별칭이 존재할 경우 컬럼명 앞에 "별칭."을 붙임
*/
select distinct st.name as "이름" 
from student st
```
