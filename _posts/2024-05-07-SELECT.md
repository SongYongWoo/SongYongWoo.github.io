select : 결과창에 보여줄 값을 지정

``` mysql
select 컬럼명 from 테이블명 ;
```

```mysql
select distinct 컬럼명  -- 중복되지 않은 속성값만 출력
	컬럼명 [as] 별칭 -- 별칭이 한글일 경우 "" 필요
from 테이블명 [as] 별칭 ;

select distinct st.name as "이름" -- 테이블에 별칭이 있을 경우 컬럼명 앞에 별칭. 을 입력.
from student as st
```



desc : 테이블 내 컬럼명과 타입을 출력

```my
desc 테이블명;
```
